تجهيز حسابات المطور والنشر
=========================

مهم جدًا
--------
أنا جهزت لك ملف المطور والهيكل، لكن إنشاء حسابات فعلية باسمك على الخدمات الخارجية يحتاج بياناتك الشخصية ووسيلة دفع وبريدك، لذلك لا يمكن إنشاؤها تلقائيًا بدونك.
الملف ده يختصر لك كل الحسابات المطلوبة وما الذي تجهزه داخل كل حساب.

الحسابات الأساسية المطلوبة
--------------------------
1) GitHub
- لحفظ المشروع وإدارة الإصدارات
- ارفع مجلد اللعبة كمستودع جديد

2) Google Play Console
- لنشر نسخة Android
- يحتاج حساب مطور مدفوع مرة واحدة

3) Apple Developer
- لنشر نسخة iPhone/iPad
- يحتاج اشتراك سنوي

4) Firebase
- تسجيل دخول
- قاعدة بيانات
- تحليلات
- إشعارات Push

5) AdMob
- للإعلانات وتحقيق الدخل

6) Cloudflare أو Vercel أو Netlify
- لاستضافة نسخة الويب

7) Unity Asset Store أو Unreal Marketplace (اختياري)
- لو هتنقل المشروع لإصدار 3D لاحقًا

بيانات لازم تجهزها قبل إنشاء الحسابات
-------------------------------------
- اسم المطور أو اسم الاستوديو
- بريد احترافي
- رقم هاتف
- دولة الحساب
- عنوان كامل
- وسيلة دفع
- سياسة خصوصية
- أيقونة اللعبة
- صور المتجر
- وصف اللعبة
- تصنيف الأعمار

اقتراح اسم نشر جاهز
--------------------
Studio Name: FireZone Interactive
Game Name: FireZone Royale
Short Description: Fast battle royale action with shrinking safe zone and intense survival gameplay.

خطوات نشر نسخة الويب
--------------------
1) ارفع الملفات إلى GitHub
2) اربط المستودع مع Vercel أو Netlify
3) اجعل ملف index.html هو نقطة البداية
4) فعّل دومين لو حبيت

خطوات نشر Android
------------------
1) لفّ المشروع باستخدام Capacitor
2) افتحه داخل Android Studio
3) أضف أيقونات واسم الحزمة
4) أنشئ ملف APK أو AAB
5) ارفعه على Google Play Console

خطوات نشر iPhone
-----------------
1) لفّ المشروع باستخدام Capacitor
2) افتحه داخل Xcode على macOS
3) أضف Bundle Identifier
4) اربطه بحساب Apple Developer
5) ارفع البناء على App Store Connect

ملفات تسويقية أنصحك تجهزها
---------------------------
- icon.png
- cover.jpg
- screenshots/
- privacy-policy.html
- terms.html
- trailer.mp4

- نسخة APK/Android packaging guide
- نسخة Unity design package
- ملف سياسة خصوصية وشروط استخدام
