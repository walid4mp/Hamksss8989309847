FireZone Royale — دليل التشغيل والتحميل
=====================================

مقدمة
------
دي نسخة أصلية مستوحاة من ألعاب الباتل رويال، وليست نسخة مطابقة أو منسوخة من Free Fire.
المشروع معمول كتجربة جاهزة قابلة للعب على الويب + ملفات مطور تساعدك تبدأ التطوير والتعديل بسرعة.

محتويات المشروع
---------------
1) index.html         الواجهة الرئيسية وتشغيل اللعبة
2) styles.css         التصميم والـ HUD والـ Overlay
3) game.js            منطق اللعبة بالكامل
4) game_config.json   إعدادات قابلة للتعديل
5) DEVELOPER_ACCOUNTS_AND_PUBLISHING_AR.txt   تجهيز حسابات المطور والنشر
6) PROJECT_STRUCTURE_AR.txt                    شرح البنية الفنية

طريقة التشغيل السريعة
----------------------
الطريقة 1:
- افتح ملف index.html مباشرة في المتصفح.

الطريقة 2: الأفضل للمطورين
- شغّل سيرفر محلي بسيط داخل مجلد المشروع.
- لو عندك Python:
  python -m http.server 8080
- بعد كده افتح:
  http://localhost:8080

طريقة اللعب
-----------
- الحركة: W A S D أو الأسهم
- التصويب: الماوس
- الإطلاق: كليك شمال
- استخدام الإسعاف: E
- إعادة التذخير: R
- الهدف: ابقَ آخر لاعب حي داخل المنطقة الآمنة

أفكار التطوير القادمة
---------------------
- إضافة أسلحة متعددة
- نظام سكنات وشخصيات
- لوت أكثر تنوعًا
- مؤثرات صوتية وموسيقى
- طور أونلاين حقيقي عبر WebSocket / Photon / Nakama
- شاشة متجر وإعدادات كاملة

تحويلها لمشروع موبايل
---------------------
1) أسهل طريقة:
- غلف المشروع كتطبيق Android/iOS باستخدام Capacitor أو Cordova أو Tauri Mobile.

2) الطريقة الاحترافية:
- إعادة بناء المنطق على Unity أو Unreal مع نفس الـ Game Design.

ملحوظات مهمة
------------
- النسخة الحالية Single Player ضد Bots.
- المشروع مناسب كبداية قوية أو عرض تجريبي سريع.
- لو عايز بعد كده أقدر أكمّلك نسخة مطور أكبر فيها متجر، تسجيل دخول، سكنات، سيرفر، ولوبي.
