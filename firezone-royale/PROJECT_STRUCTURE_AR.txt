شرح بنية المشروع
================

نظرة عامة
---------
المشروع الحالي مبني كلعبة ويب خفيفة باستخدام HTML + CSS + JavaScript بدون مكتبات خارجية.
وده يخليه سهل التشغيل، سهل التعديل، وسهل التحويل لاحقًا إلى تطبيق موبايل أو إعادة بنائه على محرك ألعاب احترافي.

الطبقات الرئيسية
----------------
1) العرض Rendering
- موجود داخل game.js
- مسؤول عن رسم الخريطة واللاعب والخصوم والطلقات والـ HUD والميني ماب

2) منطق اللعب Gameplay Logic
- حركة اللاعب
- ذكاء الخصوم
- التصويب والرصاص
- المنطقة الآمنة
- اللوت والعلاج والدرع

3) الواجهة UI
- index.html + styles.css
- شاشة البداية
- شاشة النهاية
- شريط الصحة والدرع والذخيرة

4) الإعدادات Config
- game_config.json
- لتعديل القيم الأساسية بسرعة

كيف توسع المشروع
----------------
- أضف system/weapon.js لو حبيت تفصل منطق الأسلحة
- أضف system/network.js لو هتدخل أونلاين
- أضف assets/ للصور والمؤثرات الصوتية
- أضف auth/ و backend/ لو هتعمل حسابات لاعبين

اقتراح بنية احترافية مستقبلًا
-----------------------------
/src
  /core
  /entities
  /systems
  /ui
  /assets
  /network
  /audio
  /config

تحويلها إلى 3D
--------------
لو هدفك لعبة أقرب جدًا لمستوى الألعاب الكبيرة:
- استخدم Unity أو Unreal
- انقل نفس عناصر التصميم: الخريطة، اللوت، المنطقة، الأسلحة، الواجهة
- أضف أنيميشن، مؤثرات، وماتش ميكنج وسيرفرات
